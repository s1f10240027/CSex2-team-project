## class7-team5-project
### タイトル　
「MusiQ」 - "Music Questions"の略  
![image](/django1/static/media/logo.png)
### 概要
MusiQは、音楽をテーマにしたクイズ形式のWebアプリケーションです。このアプリは「イントロクイズ」から着想を得て開発されました。楽曲の一部分を聞き取り、そのタイトルを当てることが目的です。  
  
特徴的として、再生箇所が楽曲の冒頭（イントロ）に限定されないため、幅広い音楽知識が求められます。  
音楽好きな方から、クイズで挑戦したい方まで、楽しめる内容となっています。  

> [!IMPORTANT]
> 本アプリにて搭載している音楽素材は、教育目的で利用しており担当先生の承諾を得た上で使用しています。

### 開発環境
プログラミング言語: Python, HTML, CSS, JavaScript  
フレームワーク: Django  
データベース: SQLite
