## class7-team5-project
### アプリケーションについて：
#### タイトル　
「MusiQ」 ("Music Questions"の略)  
![image](/django1/static/media/logo.png)
#### 概要
MusiQ は、音楽を題材としたクイズ形式のwebアプリケーションです。
本アプリは「イントロクイズ」から発想を得ており、楽曲の一部分を再生し、その音楽が何であるかを当てることを目的としたアプリとなっています。
再生される箇所は必ずしも楽曲の冒頭（イントロ）ではなく、曲中のランダムな部分であるため、幅広い音楽知識が求められます。

> [!IMPORTANT]
> 本アプリにて搭載している音楽素材は、教育目的で利用しており担当先生の承諾を得た上で使用しています。

### 開発環境
プログラミング言語: Python, HTML, CSS, JavaScript  
フレームワーク: Django  
データベース: SQLite
